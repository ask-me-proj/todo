# API

## Description

Here we build api for todo

## Installation

```shell
yarn -i
```

## Running the app

```shell
# runs server-side app
yarn workspace api dev

# runs whole app
yarn dev
```

## Testing

```shell
# unit tests
yarn workspace api test

# e2e tests
yarn workspace api test:e2e

# test coverage
yarn workspace api test:cov
```
